How to use
  1. Host this page — Upload video-qr.html to any web server or use GitHub Pages (free, global CDN, 5 minutes to set up).
  2. Paste the hosting URL above (e.g. https://username.github.io/video-qr) and update it whenever you redeploy.
  3. Paste a video URL — YouTube, Vimeo, or a direct MP4 link hosted on your own server/CDN.
  4. Generate & download the QR code as a PNG image.
  5. Print or share — anyone scanning the code with their phone will instantly see the video.
Global access tip: For truly worldwide reach (including regions where YouTube is blocked), host the video as a direct MP4 file on a CDN (Cloudflare R2, AWS S3 + CloudFront, BunnyCDN, etc.).
